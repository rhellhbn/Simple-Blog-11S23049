About Me

Halo, saya Rahel Debora Hasibuan, seorang mahasiswa informatika yang sedang belajar pengembangan web. Saat ini saya aktif mendalami berbagai teknologi modern yang mendukung pembangunan aplikasi berbasis web dan mobile.

Saya memiliki minat besar pada pemrograman, machine learning, dan pengembangan aplikasi web. Selain itu, saya juga tertarik mempelajari bagaimana desain antarmuka yang baik dapat meningkatkan pengalaman pengguna (user experience). Dengan kombinasi antara logika pemrograman dan kreativitas desain, saya berharap bisa menghasilkan aplikasi yang bermanfaat bagi banyak orang.

Di luar akademik, saya juga senang mengikuti komunitas teknologi, berdiskusi dengan teman-teman sejurusan, dan berpartisipasi dalam proyek kolaboratif. Kegiatan ini membantu saya untuk terus berkembang serta memahami tren terbaru di dunia IT.

Proyek ini dibuat sebagai bagian dari studi kasus menggunakan Bootstrap 5.3. Tujuannya adalah untuk mengasah keterampilan saya dalam menggunakan framework CSS modern, membangun tampilan web yang responsive, serta memahami bagaimana struktur HTML dapat dipadukan dengan komponen Bootstrap agar lebih rapi, cepat, dan konsisten.
